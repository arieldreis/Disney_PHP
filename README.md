# Disney+ Inspired Login Screen

## 📌 Sobre o Projeto
Este projeto consiste em uma **tela de login inspirada na interface do Disney+**, desenvolvida com foco em **design moderno, experiência do usuário e prática de desenvolvimento front-end**.

O objetivo foi recriar uma interface semelhante à plataforma de streaming para praticar conceitos de **PHP, MySQL**, além de trabalhar com **estilização personalizada e layout responsivo**.

---

## 🚀 Funcionalidades
- Interface inspirada no Disney+
- Campos de login (email e senha)
- Estilização moderna com CSS
- Uso de fontes personalizadas
- Estrutura organizada para integração futura com back-end
- Layout responsivo

---

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Fontes personalizadas
- PHP
- MySQL

---

## 🎯 Objetivo do Projeto
Este projeto foi criado com o objetivo de:

- Praticar **criação de interfaces modernas**
- Treinar **organização de código front-end**
- Preparar a base para integração com **sistemas de autenticação**
- Melhorar habilidades em **design de UI**
